# Tin
Luanti server manager tool
rough around the edges but it works this is just the alpha version
posting src after a few more updates
note this was only tested on debian based linux flavors not windows

installation proccess:
```chmod +x Tin-v1.0-linux-x86_64.AppImage```
do this in the terminal while in the same dir

then:
```./Tin-v1.0-linux-x86_64.AppImage```

and voila! there you have it
